# 2. Vue.js 와 django를 사용한 Todo List 만들기

프론트는 `Vue.js`, 백엔드는 `django Rest Framework(DRF)`를 사용하여 todo list 앱을 만들어보자.

## Demo

https://caesiumy.github.io/vue-django-stagram/

> DRF와의 통신 없이 오직 Vue로만 되어있어서 제대로 된 동작은 하지 않음

## 기능

- [x] GET 으로  데이터 받아오기
- [x] PUT 으로  데이터 쓰기
- [x] DELETE 로 데이터 삭제하기
- [x] PUT 으로  데이터 수정하기
